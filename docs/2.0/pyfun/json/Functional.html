5.2.7. Options for Functional Section¶

The options below are the available options in the Functional Section of the pyfun.json control file

Functions: {None} | dict

value of option “Functions”

Coeffs: {None} | dict

value of option “Coeffs”

5.2.7.1. Options for all Coeffs¶

compID: {None} | int | int8 | int16 | int32 | int64 | uint8 | uint16 | uint32 | uint64 | str

value of option “compID”

target: {0.0} | float | float32

value of option “target”

parent: {None} | str

value of option “parent”

weight: {1.0} | float | float32

value of option “weight”

power: {1.0} | float | float16 | float32 | float64 | float128 | int | int8 | int16 | int32 | int64 | uint8 | uint16 | uint32 | uint64

value of option “power”

5.2.7.2. Options for all Functions¶

coeffs: {None} | list[str]

value of option “coeffs”

parent: {None} | str

value of option “parent”

type: {'adapt'} | 'constraint' | 'objective'

value of option “type”
