Usage:
$ cape-tec LAYOUT [O] [OPTIONS]
Arguments:

LAYOUT: Name of Tecplot(R) layout file to use O: Output file (default based on layout with changed extension)

Options:
--clean

Delete macro after export (use --no-clean to suppress)

--ext, --fmt, --format EXT

Export format {PNG}

--help, -h

Display this help message and exit

--layout, --lay LAY

Name of Tecplot(R) layout file to use

-o, --output FNAME

Output file (default based on layout with changed extension)

--verbose, -v

Increase verbosity during process

--width, -w WIDTH

Image width, in pixels
