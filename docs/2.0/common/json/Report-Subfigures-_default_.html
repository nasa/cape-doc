3.1.9.2.1. Options for default subfigure¶

Option aliases:

  • Parent -> Type

  • parent -> Type

  • pos -> Position

  • type -> Type

  • width -> Width

Recognized options:

Alignment: {'center'} | object

value of option “Alignment”

Caption: {None} | str

subfigure caption

Position: {'b'} | 'c' | 't'

subfigure vertical alignment

Type: {None} | str

subfigure type or parent

Width: {0.33} | float

value of option “Width”
