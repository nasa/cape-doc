cape.lava: Interfaces to LAVA from NASA Ames¶

cape.pylava: CAPE support for NASA’s LAVA solver suite¶

This module provides interfaces to use CAPE’s capabilities with the LAVA suite of CFD solvers created by the Computational Aerosciences branch at NASA Ames Research Center.

  • cape.pylavacntl: LAVA control module
  • cape.pylava.options: Options interface for LAVA
    • cape.pylava.options: Options interface for cape,pylava
    • cape.pylava.options.lavaopts: Options for Superlava executable
    • cape.pylava.options.meshopts: LAVACURV mesh options
    • cape.pylava.options.runctlopts: LAVACURV run control options
    • cape.pylava.options.util: Utilities for pyLava options module
  • cape.pylava.case: LAVACURV case control module
  • cape.pylava.cmdgen: Create commands for LAVA executables
  • cape.pylava.dataiterfile: Interface to LAVA data.iters files
  • cape.pylava.report: Automated report interface
  • cape.pylava.yamlfile: Custom class for LAVA input YAML files
