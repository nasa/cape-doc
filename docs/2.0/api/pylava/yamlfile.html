cape.pylava.yamlfile: Custom class for LAVA input YAML files¶

This module provides the class RunYAMLFile that interacts with the YAML (Yet Another Markup Language) files used to control settings for the various LAVA solvers.

class cape.pylava.yamlfile.RunYAMLFile(*args, **kw)¶

Custon YAML file control class

Call:
>>> obj = RunYAMLFile(fname=None)
Inputs:
fname: {None} | str

Name of file to read

Outputs:
obj: RunYAMLFile

YAML file interface instance
