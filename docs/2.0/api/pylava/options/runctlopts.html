cape.pylava.options.runctlopts: LAVACURV run control options¶

This is the "RunControl" options module specific to cape.pylava. It is based primarily on

cape.cfdx.options.runctlopts

and provides the class

cape.pylava.options.runctlopts.RunControlOpts

Among the LAVA-specific options interfaced here are the options for the LAVA executables, defined in the "RunControl" section
