cape.filecntl.texfile: Interface to LaTeX source files¶

This is a module built off of the cape.filecntl.FileCntl class customized for manipulating Cape’s automated reports. In addition to containing methods for reading, writing, and organizing lines of text, it contains a Compile() method for compiling the PDF and sparing the user from constructing the system command to do so.

See the cape.cfdx.report module or the JSON report page page for more information.

File control class for LaTeX files
