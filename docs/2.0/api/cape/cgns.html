cape.cgns: CGNS file interface¶

This module provides a class for reading CGNS files of certain types that have been implemented. The main purpose is to read surface triangulations with quads and convert it into a cape.trifile.Tri object. However, the class provided in this module, cape.cgns.CGNS, must be converted into a cape.trifile.Tri or other object externally. This module merely reads the file, reads the data from each node, and constructs a sub-node table.

In order to create a triangulation interface from a CGNS file, see cape.trifile.Tri, and use the cgns keyword argument.

cape.cgns.ADFAddress2Pos(addr)¶

Convert ADF 12-byte code into position index

This is a hex code with but skipping the 8th character

Call:
>>> pos = ADFAddress2Pos(addr)
Inputs:

12-character string hex code

Outputs:

Position in file in bytes

ReadADFData(f, dt)¶

Read one DaTa entry from an open CGNS/ADF file

The next four bytes must be the string "DaTa", and the following 12 bytes must give the address of the end of the field as a hex code string.

Call:
>>> data = cgns.ReadADFData(f, dt)

cgns: cape.cgns.CGNS

CGNS file interface
