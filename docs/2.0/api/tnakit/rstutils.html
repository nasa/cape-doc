cape.tnakit.rstutils: Tools for writing ReST files¶

This module contains tools to write text formatted for reStructuredText (reST). This includes various markup options and tools for creating the directives to include images. Of particular interest are the functions rst_image_table_lines() and rst_image_table(), which simplifies the important task of creating a table that includes several images.

cape.tnakit.rstutils.rst_title(title, char='-', margin=2, before=False)¶

Create an ReST title from string

This adds characters before (optionally) and after the title, creating valid syntax for ReStructuredText titles. Examples of valid titles with the input string "Sample Title" are shown below:

Sample Title
--------------

==============
Sample Title
==============

************
Sample Title
************
Call:
>>> txt = rst_title(title, char="-", margin=2, before=False)
Inputs:
title: str | unicode

Input string, must be one line

char: {"-"} | "=" | "*" | "^"

Valid reST title adornment character (there are more than 4)

margin: {2} | int >= 0

Extra characters to place after end of character

before: True | {False}

Whether or not to use and adornment line before title

Outputs:
txt: str | unicode

String with title, adornment line(s), and 2 or 3 newline chars
