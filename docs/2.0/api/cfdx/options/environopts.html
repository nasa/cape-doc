cape.cfdx.environopts: Options for environment variables¶

This module provides the class EnvironOpts, which controls options for environment variables that users may wish to set at the beginning of each individual case in a run matrix.

class cape.cfdx.options.environopts.EnvironOpts(*args, **kw)¶

Class for environment variables

Call:
>>> opts = EnvironOpts(**kw)
Inputs:

Dictionary of environment variables
