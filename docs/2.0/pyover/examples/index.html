6.1. Examples using pyOver¶

Example projects that demonstrate how to use pyCart can be found in $CAPE/examples/pyover/ where $CAPE is the folder in which Cape is installed. The examples are intended to roughly increase in capability in an interactive fashion to initiate users to the various capabilities of pyOver and OVERFLOW.
