6.2.8. Options for PBS Section¶

The options below are the available options in the PBS Section of the pyover.json control file

r: {'n'} | 'y'

rerun-able setting

W: {''} | str

PBS W setting, usually for setting group

ncpus: {128} | int | int32 | int64

number of cores (roughly CPUs) per node

q: {'normal'} | str

PBS queue name

model: {'rom'} | str

model type/architecture

S: {'/bin/bash'} | str

shell to execute PBS job

walltime: {'8:00:00'} | str

maximum job wall time

aoe: {None} | str

architecture operating environment

p: {None} | object

PBS priority

A: {None} | object

account name(s) or number(s)

mpiprocs: {128} | int | int32 | int64

number of MPI processes per node

o: {None} | object

explicit STDOUT file name

select: {1} | int | int32 | int64

number of nodes

ompthreads: {None} | int | int32 | int64

number of OMP threads

e: {None} | object

explicit STDERR file name

j: {'oe'} | str

PBS ‘join’ setting
