cape.pyfun.faux: FAUXGeom interface module¶

This is a module for interacting with FUN3D input files that define geometry for adaptation. Files defined in the FAUXGeom file faux_input can have their surface meshes refined while other surfaces must be frozen.

See also:

  • cape.pyfun.cntl.Cntl.ReadFAUXGeom()

  • cape.pyfun.cntl.Cntl.PrepareFAUXGeom()

  • cape.pyfun.cntl.Cntl.ReadFreezeSurfs()

  • cape.pyfun.cntl.Cntl.PrepareFreezeSurfs()

class cape.pyfun.faux.FAUXGeom(fname='faux_input')¶

File control class for faux_input

Name of faux_input file or template

Outputs:

Dictionary of geometry definitions
