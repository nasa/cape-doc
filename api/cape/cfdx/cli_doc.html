cape.cfdx.cli_doc: Template help messages for CAPE executablesÂ¶

This module provides a template docstring template that can be used as a template for each of the executables (e.g. pyfun and pycart).

This is the template for what users see when they execute

$ pyfun -h

or

$ pyover --help
